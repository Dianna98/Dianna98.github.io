After the manual data cleaning, I converted the Excel tables to CSV and validated the relationships between them using python and the pandas library. Below is the list of validations performed, all of which have passed.

  • Sample Size
    • For each industry, the sum between the number of surveys sent out for workforce size under 250 and for workforce size of 250 and over should be equal to the total specified.
    • The total number of surveys sent out for all given industries per workforce size should be at least the sum of the number of surveys sent out for each industry.
    • For the BICS survey broken down by workforce size, workforce size 0-99 + workforce size 100-249 = the total for all industries with workforce size < 250 from the previous table.
    • Workforce size 0-99 + workforce size 100-249 + workforce size 250+ = total number of surveys sent out for all industries and all size bands.
  • Response Rates
    • The checks applied for Sample Size.
    • The data from the table ‘Proportion of responses to the BICS survey broken down by industry’ should be in accordance with the rapport from the tables ‘Responses to the BICS survey broken down by industry’ and ‘Sample for the BICS survey broken down by industry’.
  • Trading Status
    • For all three tables, the values from the 3 data columns should sum up to 100% ± 0.3%, to account for potential errors introduced by rounding.
  • Government Schemes
    • For the first table, the number of companies from all industries (calculated as total number of responses * percentage of industries which continue or have temporarily paused trading * percentage of companies which have applied for certain initiative) should be at least the sum of the number of companies for each industry (calculated in the same way).
    • For the second table, using the formula presented in the previous point, the sum between workforce sizes should be approximately equal to the total. A small margin is allowed due to possible errors introduced by rounding.
  • Government Schemes (2)
    • The checks described for Government Schemes.
    • For each industry, the percentage of received initiatives should be less than the percentage applied for.
  • Government Schemes (3)
    • The checks described for Government Schemes.
